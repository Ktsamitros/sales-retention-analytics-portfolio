# 📊 Customer Transactions Analytics (Power BI Project)

## 📌 Overview
This project analyzes two full years of telecom customer transactions (2024–2025), covering Mobile, Internet/Fiber, and TV services.  
The dataset includes renewals, new contracts, upsells, regions, agents, revenue, and customer segments.  
It is fully cleaned, structured, and optimized for Power BI analytics.

---

## 📁 Dataset Summary
The dataset contains the following fields:

- Date  
- Customer Name  
- Region  
- Sales Agent  
- Product Category  
- Product Name  
- Units  
- Revenue  
- Contract Type (Renewal, New Contract, Upsell)  
- Customer Segment (Consumer, Small Business)

The data is stored in a UTF‑8 CSV file and loaded into Power BI for modeling and dashboard creation.

---

## 📊 Project Goals
- Build a complete analytics model for telecom customer transactions  
- Create KPIs for revenue, renewals, upsells, and customer segments  
- Analyze performance by region, agent, product category, and time  
- Prepare a professional Power BI dashboard for portfolio use  

---

## 🧩 Power BI Model
The Power BI file includes:

- Clean fact table: **Customer Transactions**  
- Date intelligence columns (Year, Month, Quarter, Year‑Month)  
- DAX measures for revenue, units, contract types, and time intelligence  
- Ready structure for dashboard visuals and executive reporting
