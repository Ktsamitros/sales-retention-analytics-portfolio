# 📊 Sales & Customer Analytics Portfolio (Power BI Projects)

## 📌 Overview
This repository contains a collection of professional Power BI analytics projects focused on telecom sales, customer behavior, retention, and performance insights.  
Each project includes a clean dataset, a structured Power BI model, and a documented analytics workflow.

The purpose of this portfolio is to demonstrate strong capabilities in:
- Data modeling  
- KPI development  
- DAX calculations  
- Dashboard design  
- Business insights for sales & customer analytics
